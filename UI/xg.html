Gradient Boosting-XGBoost

XGBoost is short for Extreme Gradient Boosting. It is based on gradient boosting framework but it is designed to be highly efficient and flexible. The parallel tree boosting provided by XGBoost enables this algorithm to solve data science problems with high efficiency and accuracy. XGBoost has been one of the most popular algorithms used in Kaggle competitions.

Factorization Machine

Factorization Machine is a well-used model in developing recommender systems. Factorization machine can be compared to support vector machine with a polynomial kernel. Comparing to svm, which requires dense data, factorization machine works well on sparse data. Since our data consists of significant sparsity, factorization machine is an ideal candidate to tackle the challenge

We achieved an accuracy rate of 0.465 by using Model 2.
