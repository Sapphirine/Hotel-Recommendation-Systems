Description of Data Variables

USERS INFO

# Variable Name Data Type Description
1 Date_time String Timestamp
2 Site_name int ID of the Expedia point of sale
3 Posa_cotinent Int ID of continent associated with site_name
4 User_location_country Int ID of continent associated with site_name
5 User_location_region Int The ID of the region the customer is located
6 User_location_city Int The ID of the city the customer is located
7 User_id Int ID of user
8 Is_mobile Tinyint 1 when a user connected from a mobile device,0 otherwise
9 Is_package Int 1 if the click/booking was generated as a part of a package,0 otherwise
10 Channel Int ID of a marketing channel
11 Cnt Bigint Numer of similar events in the context of the same user session
12 Hotel_cluster Int ID of a hotel cluster


TRIP DETAILS

# Variable Name Data Type Description
1 Orig_destination_distance Double Physical distance between a hotel and a customer at the time of search.
2 Srch_ci String Checkin date
3 Srch_co String Checkout date
4 Srch_adults_cnt Int The number of adults specified in the hotel room
5 Srch_children_cnt Int The number of (extra occupancy) children specified in the hotel room
6 Srch_rm_cnt Int The number of hotel rooms specified in the search
7 Srch_destination_id Int ID of the destination where the hotel search was performed
8 Srch_destination_type_id Int Type of destination


HOTEL INFO

# Variable Name Data Type Description
1 Hotel_continent Int Hotel continent
2 Hotel_country Int Hotel country
3 Hotel_market Int Hotel market


DECISION

# Variable Name Data Type Description
1 Is_booking Tinyint 1 if a booking, 0 if a click
