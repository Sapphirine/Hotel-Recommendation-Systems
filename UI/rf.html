Random Forest

Random forest is an ensemble learning method that is applicable for both regression and classification problems. By randomly selecting a given number of features each time, comparing to simple tree-based methods, random forest can reduce variance and decorrelates trees. In this case, random forest is a good choice on our dataset, which consists of 25 variables.

Popular Ranking

The popularity ranking algorithm is based on a simple logic that if we assume travelers share similar preference, the best recommendations we can make is to choose the most popular hotels in the area. With the algorithm, we count the bookings of each hotel and rank the hotels in each area by number of booking. For each search, we present the top five most popular hotels in the customer destination.

Popularity = the number of booking + click weight (0.05) * the number of click
